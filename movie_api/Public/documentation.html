Welcome to a Cinema API built on REST.
It returns JSON objects as responses

GET /movies

Returns a list of all movies.

Parameters

None

Example Request

    GET /movies HTTP/1.1
    Host: example.com

Add a user /users POST A JSON object holding data about the user to add, structured like:

{ 'Name': 'John Doe',
'Username': '[username]',
'Email': [email@example.com],
'Birth-date': '1980-1-31' } A JSON object holding data about the user that was added, including their assigned id:
{ '_id': '######',
'Name': 'John Doe',
'Username': '[username]',
'Email': [email@example.com],
'Birth-date': '1980-1-31' } Add a movie object to a user's favorites list

When a PUT request is made to /users/:username/movies/:movieID, you would look up the user in your database using the username provided in the URL parameters (req.params.username). Then, you would add the movie ID provided in the URL parameters (req.params.movieID) to that user's list of favorite movies in the database.

/users/[username]/movies/[movieID] PUT None A text message indicating whether the movie was successfully added from the user's favorited movies Remove a movie from a user's favorites list /users/[username]/movies/[movieID] DELETE None This movie has been removed from favourites.
